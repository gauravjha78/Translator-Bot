# Translator-Bot
An intuitive, mobile-responsive web app (or bot) designed to break down language barriers. It takes English text and translates it instantly into Hindi, Marathi, or Spanish. With integrated Text-to-Speech (TTS) functionality, users can not only read the translations but also hear the accurate pronunciation of the output in real-time.
